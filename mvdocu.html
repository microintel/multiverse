Multiverse

Secure your data with ease using our versatile encryption-decryption library.

Introduction

The Multiverse, is Cryptography Library
written in JavaScript
developed by Microintel designed to provide robust encryption and decryption capabilities for web applications.
This library supports a wide range of character sets, making it highly versatile and adaptable for various use cases.

Installation

Include the Multiverse Encryption-Decryption Library in your project by adding the following script tag:

<script src="https://microintel.github.io/multiverse/multiverse.js"></script>

Parameters

  • 1st parameter : [for user input] input text to be encrypted or decrypted.
  • 2nd parameter : The operation to be performed: 'e' for encryption, 'd' for decryption.
  • 3rd parameter : The key for encryption/decryption.
  • 4th parameter : The type of character set to be used. Refer to the character sets section for available types.here

Usage

Basic Usage

const plaintext = 'hello';
                   const key = 5;
                      const charSet = 'alpha';

                  const encryptedText = ed(plaintext, 'e', key, charSet);
                  console.log('Encrypted Text:', encryptedText);

          const decryptedText = ed(encryptedText, 'd', key, charSet);
    console.log('Decrypted Text:', decryptedText);

Character Sets

The library supports multiple character sets, each suitable for different purposes. Here are the available types:

  • abcd : Alphabetic characters only (uppercase and lowercase). 
    const encryptedText = ed('hello', 'e', 3, 'abcd');
  • alpha : Alphanumeric characters, including special symbols. 
    const encryptedText = ed('hello123!', 'e', 5, 'alpha');
  • spc : Special symbols only. 
    const encryptedText = ed('!@#$', 'e', 4, 'spc');
  • kalpha : Kannada + alphabetic characters. 
    const encryptedText = ed('ಹೆಲ್ಲೋ', 'e', 2, 'kalpha');
  • telpha : Telugu + alphabetic characters. 
    const encryptedText = ed('హలో', 'e', 6, 'telpha');
  • galpha : Greek + alphabetic characters. 
    const encryptedText = ed('γειά', 'e', 7, 'galpha');
  • talpha : Tamil + alphabetic characters. 
    const encryptedText = ed('வணக்கம்', 'e', 3, 'talpha');
  • malpha : Malayalam + alphabetic characters. 
    const encryptedText = ed('ഹലോ', 'e', 5, 'malpha');
  • nalpha : Hindi + alphabetic characters. 
    const encryptedText = ed('नमस्ते', 'e', 4, 'nalpha');
  • allpha : A combination of all supported character sets. 
    const encryptedText = ed('Hello123!', 'e', 8, 'allpha');

Note

Some keys may not work properly due to the nature of the character sets. Please test different keys to find a working key for your specific use case.
